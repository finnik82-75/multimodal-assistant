# Multimodal Telegram Assistant (RAG + Voice + Vision)

## 📌 О проекте

Телеграм-ассистент для отдела продаж «Забайкальская медиа группа» (ЗМГ). Назначение — автоматизация первичного контакта с рекламодателями: квалификация запроса, консультация по форматам размещения (радио, портал ZAB.RU, ЗабТВ, соцсети) и подведение к записи на консультацию с менеджером.

Реализованы текстовый диалог, режим RAG по базе знаний с указанием источников, голосовой ввод и ответ (STT/TTS), анализ изображений (Vision). Пайплайн построен вокруг единого роутера: handlers → router → OpenAI + ChromaDB → ответ в Telegram.

## 🎯 Бизнес-задача

- Квалификация входящих запросов
- Консультация по рекламным форматам
- Перевод в менеджерскую воронку
- Работа без раскрытия прайса

## 🚀 Бизнес-результат

- Снижение нагрузки на менеджеров
- Единая точка входа для рекламодателей
- Масштабируемая консультация через RAG
- Стандартизация коммуникации

## 🏗 Архитектура

Пайплайн:

**User → handlers → router → OpenAI (LLM / Vision / STT / TTS) + ChromaDB → Telegram response**

- **handlers** — приём сообщений (текст, голос, фото, документы) и команд, вызов router
- **router** — выбор режима, сборка контекста, вызов LLM или RAG
- **rag** — загрузка и чанкинг документов, индексация в ChromaDB, поиск по эмбеддингам, ответ с источниками
- **services** — openai_client, stt, tts, vision

## 🧠 Режимы работы

| Режим | Описание |
|-------|----------|
| text | Диалог-консультант |
| rag | Ответы строго по базе знаний |
| voice | Голосовой ввод (Whisper), ответ текстом/голосом |
| vision | Анализ изображений |

Переключение: `/mode text | rag | voice | vision`

## 📚 RAG-механика

- Загрузка документов (PDF / TXT / MD)
- Чанкинг
- Индексация в ChromaDB
- Поиск по эмбеддингам
- Ответ с указанием источников
- Проверка статуса через `/stats`

## 🛠 Технологии

- Python
- pyTelegramBotAPI (AsyncTeleBot)
- OpenAI API (GPT-4o, Whisper, TTS, Vision)
- LangChain
- ChromaDB
- OpenAI Embeddings

## ⚙ Как запустить

```bash
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

Создать `.env`:

```
TELEGRAM_BOT_TOKEN=
OPENAI_API_KEY=
```

Запуск:

```bash
python main.py
```
